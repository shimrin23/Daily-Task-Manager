#  TaskManager (WPF + Web Assets)

**TaskManager** is a simple yet functional WPF desktop application built in **C#**, designed to manage to-do tasks.  
The project also includes several **web-based demo pages** for login, task management, and notes, providing both desktop and browser interfaces.

##  Overview

**TaskManager** helps users create, view, and manage their to-do tasks in a simple desktop interface.  
In addition to the WPF app, this workspace includes several **HTML/CSS/JS demo frontends** that mirror core app features such as login, notes, and task lists.
